Breeze Cli

简化流程的开发工具

快速上手 →

功能丰富

对 TypeScript、ESLint、PostCSS 提供开箱即用的支持。

开发简洁

提供 Nunjuck 模板语法和开发热重载，可以快速开发项目

构建更舒心

提供图片压缩、内联 style 和内联 script polyfill等功能

# 起步

安装：

npm install -g @boses/breeze-cli
# OR
yarn global add @boses/breeze-cli

创建一个项目：

breeze created my-project
